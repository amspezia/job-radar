# Job Radar

An **observable, evaluated agentic system** that finds remote developer roles, drafts
tailored applications under human approval, and tracks them to outcome — exposed over MCP.
The centerpiece is reliability: evaluation, observability, human-in-the-loop control, and
guardrails. Retrieval is the substrate, not the headline.

> **Status:** scaffold / WIP
